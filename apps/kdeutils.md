# KDEUTILS

 This metapackage includes general-purpose utilities 
 provided with the official release of KDE:
 
 - ark
 - filelight
 - kcalc
 - kcharselect
 - kdf
 - kgpg
 - ktimer
 - kwalletmanager5
 - sweeper
 
 SITE: https://apps.kde.org
